# VM1 Configuration - No Firewall (Baseline)

## Network Configuration

### File: `/etc/netplan/01-netcfg.yaml`

```yaml
network:
  version: 2
  ethernets:
    enp0s3:       # Host-only (Kali)
      dhcp4: no
      addresses: [192.168.56.10/24]
      nameservers:
        addresses: [8.8.8.8]
    enp0s8:       # NAT Network (Internet)
      dhcp4: true
```

### Apply Configuration:
```bash
sudo netplan apply
```
